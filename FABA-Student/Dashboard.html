Financial Aid Criteria

  • ✔ Must be enrolled in an accredited institution.
  • ✔ Family income below the required threshold.
  • ✔ Minimum academic performance required.
  • ✔ PRN number required.
  • ✔ Aid is provided based on sponsor contributions.

Book Aid Criteria

  • ✔ Must be a student in a recognized institution.
  • ✔ Proof of enrollment required.
  • ✔ Provide details of required books.
  • ✔ Request must be within donation limits.
  • ✔ Students can donate books to others.

About FABA

FABA (Financial Aid & Book Aid) is a platform that connects students, sponsors, and admins to manage financial assistance and book donations efficiently.

Key Features:

  • 🔹 Apply for financial aid & track status.
  • 🔹 Request & donate books.
  • 🔹 Sponsors can contribute directly.
  • 🔹 Secure platform for aid requests.
